## Europäische Gesellschaft (SE)

Die **Europäische Gesellschaft (Societas Europaea, SE)** ist eine **europäische Rechtsform für Kapitalgesellschaften**.  
Sie ermöglicht Unternehmen, **EU-weit unter einer einheitlichen Rechtsform tätig zu sein**, ohne für jedes Land eine eigene Gesellschaft gründen zu müssen.

## Zentrale Merkmale der SE

| Merkmal | Erklärung |
|---|---|
| Eigene Rechtspersönlichkeit | Die SE ist eine eigenständige juristische Person |
| Kapitalgesellschaft | Ähnlich aufgebaut wie eine Aktiengesellschaft (AG) |
| Mindestkapital | Mindestens **120.000 € Grundkapital** |
| Haftung | Aktionäre haften nur mit ihrer **Kapitaleinlage** |
| Organisationsstruktur | Wahl zwischen **monistischem** und **dualistischem** System |

## Organisationsmodelle der Geschäftsführung

| System | Struktur |
|---|---|
| Monistisches System | Ein Verwaltungsrat führt und überwacht gleichzeitig |
| Dualistisches System | Trennung von **Vorstand (Leitung)** und **Aufsichtsrat (Kontrolle)** |

## Vorteile der SE

- Einheitliche Rechtsform für **europäische Unternehmen**
- Erleichtert **grenzüberschreitende Unternehmensstrukturen**
- Flexible Organisationsform
- Höhere internationale Wahrnehmung

## Prüfungsrelevanz (AP1)

Typische Prüfungsanforderung:

- **Merkmale der SE nennen**

Wichtige Stichpunkte:

- Kapitalgesellschaft  
- Mindestkapital **120.000 €**  
- Haftung nur mit Einlage  
- Monistisches oder dualistisches System  

## Merksatz

> **SE = Europäische Aktiengesellschaft mit flexiblem Leitungssystem.**

## Häufige Prüfungsfalle

| Fehler | Korrektur |
|---|---|
| SE ist eine Personengesellschaft | Die SE ist eine **Kapitalgesellschaft** |
| Geschäftsführer haften persönlich | Aktionäre haften nur mit ihrer **Einlage** |
| Es gibt nur eine Leitungsstruktur | Die SE kann **monistisch oder dualistisch** organisiert sein |
